# HCI

Ovo je WPF program za sklapanje proslava

## Installation

Program nije neophodno instalirati, pokrece se pomocu .exe file-a.

Moze se i iskompajlirati iz source code-a

## Usage

Pokretanjem .exe file-a se pokrece program

Mozete se Ulogovati ako vec imate nalog, ili se mozete registrovati kao Korisnik.

Preciznija uputstva za upotrebu mozete naci u pomocnim vide-ima pustenim u samoj aplikaciji.

Takodje imamo i dostupna uputstva u HTML fajlovima.
